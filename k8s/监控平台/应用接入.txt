1.编译镜像
docker build -t 119.91.113.111:8081/docker-local/kubeblog:1.3 .

2.设置latest标签
docker tag 119.91.113.111:8081/docker-local/kubeblog:latest 119.91.113.111:8081/docker-local/kubeblog:latest

3.推送到JCR仓库
docker push 119.91.113.111:8081/docker-local/kubeblog:latest

4.部署应用
使用 /Helm/kubeblog 目录

helm package kubeblog

helm install kubeblog kubeblog-0.1.2.tgz

查看应用的metrics 监控数据接口
http://192.168.0.128:30002/actuator/prometheus
